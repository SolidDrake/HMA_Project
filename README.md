Informazioni sulla repository

Organizzazione Cartelle:

DB: La seguente cartella contiene il DB MySql del progetto più un eventuale export nel caso si volesse importare su MyPhpAdmin.
MySQLWorkbench: Contiene lo schema del Database utilizzato nel progetto.
Phonegap_project: Contiene il source code e l'apk dell'applicazione mobile Phonegap del progetto (Si noti che il progetto phonegap si è basato sulla versione sito Statico che è contenente nella cartella WEBSITE).
WEBSITE: Contiene tutto il codice sorgente del sito in modalità statica, questo è dovuto a una prima stesura del sito fatto all'inizio dell'implementazione della parte di technology del corso.
WEBSITE_DYNAMIC: è la cartella contentente tutto il codice sorgente del sito dinamico caricato sul provider Altervista. (Nel caso dovessero sorgere problemi in locale, vi suggeriamo di importare il database tramite Xamp e lanciare il sito web da li, nel caso persista ancora il problema, una versione stabile del progetto è situato al link : http://affaticatidonadiopieta.altervista.org/  , questa versione è completamente funzionante e il codice è identico a quello delle pagine all'interno della cartella in esame).


Informazioni aggiuntive:

Il layout del sito è basato su un template che supporta bootstrap chiamato Modern Business Template.
